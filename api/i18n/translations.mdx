## Overview

The `ui` translation object contains all user-facing strings for the Del Poble Pizzeria website in three languages: Spanish (es), English (en), and Valencian (val).

## Structure

The translation object is exported from `src/i18n/ui.ts` and follows this structure:

```typescript
export const ui = {
  es: { /* Spanish translations */ },
  en: { /* English translations */ },
  val: { /* Valencian translations */ }
} as const;
```

## Usage Example

Access translations directly from the `ui` object:

```astro
---
import { ui } from '@/i18n/ui';

const spanishGreeting = ui.es['form.success'];
const englishGreeting = ui.en['form.success'];
const valencianGreeting = ui.val['form.success'];
---

<p>{spanishGreeting}</p>
<!-- Output: ¡Gracias por suscribirte! Revisa tu email. -->
```

## Adding New Translations

When adding new translation keys:

1. Add the key to all three language objects (es, en, val)
2. Use dot notation for namespacing (e.g., `section.subsection.key`)
3. Ensure the key exists in `ui[typeof defaultLang]` for type safety
4. Use the `useTranslations` function to access translations in components

```typescript
export const ui = {
  es: {
    'new.section.key': 'Nuevo texto',
    // ...
  },
  en: {
    'new.section.key': 'New text',
    // ...
  },
  val: {
    'new.section.key': 'Nou text',
    // ...
  }
};
```
