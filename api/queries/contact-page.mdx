## getContactPageQuery

Retrieves all content for the contact page including highlighted text, contact image, form content, and frequently asked questions.

### Function Signature

```typescript
getContactPageQuery(lang: string = 'es'): string
```

### Parameters

<ParamField path="lang" type="string" default="es">
  Language code for localized content. Supports 'es', 'ca', 'en', etc. Falls back to Spanish if translation not available.
</ParamField>

### GROQ Query

```groq
*[_type == "contactPage" && _id == "contactPage"][0] {
  "highlightedText": {
    "desktop": coalesce(highlightedText.desktop.ca, highlightedText.desktop.es, ""),
    "tablet": coalesce(highlightedText.tablet.ca, highlightedText.tablet.es, ""),
    "mobile": coalesce(highlightedText.mobile.ca, highlightedText.mobile.es, "")
  },
  "image": {
    "url": image.asset->url,
    "alt": image.alt
  },
  "imageText": coalesce(imageText.ca, imageText.es, ""),
  "formHighlightedText": {
    "desktop": coalesce(formHighlightedText.desktop.ca, formHighlightedText.desktop.es, ""),
    "tablet": coalesce(formHighlightedText.tablet.ca, formHighlightedText.tablet.es, ""),
    "mobile": coalesce(formHighlightedText.mobile.ca, formHighlightedText.mobile.es, "")
  },
  "formBodyText": coalesce(formBodyText.ca, formBodyText.es, ""),
  "faqsHighlightedText": {
    "desktop": coalesce(faqsHighlightedText.desktop.ca, faqsHighlightedText.desktop.es, ""),
    "tablet": coalesce(faqsHighlightedText.tablet.ca, faqsHighlightedText.tablet.es, ""),
    "mobile": coalesce(faqsHighlightedText.mobile.ca, faqsHighlightedText.mobile.es, "")
  },
  "faqs": faqs[] {
    "question": coalesce(question.ca, question.es, ""),
    "answer": coalesce(answer.ca, answer.es, "")
  }
}
```

### Usage Example

```typescript
import { sanityClient } from '@/lib/sanity';
import { getContactPageQuery } from '@/lib/sanityQueries';

// In an Astro page component
const lang = Astro.currentLocale || 'es';
const contactData = await sanityClient.fetch(getContactPageQuery(lang));
```

```astro

// pages/contacto.astro
import { sanityClient } from '@/lib/sanity';

const lang = Astro.currentLocale || 'es';

---

<section>

  <div class="hero-image">
    <img src={data.image.url} alt={data.image.alt} />
    <p>{data.imageText}</p>

  <div class="contact-form">
    <h2>{data.formHighlightedText.desktop}</h2>
    <p>{data.formBodyText}</p>

  </div>
  
  <div class="faqs">
    <h2>{data.faqsHighlightedText.desktop}</h2>
    {data.faqs.map(faq => (
      <details>
        <summary>{faq.question}</summary>
        <p>{faq.answer}</p>
      </details>
    ))}
  </div>
</section>
```
