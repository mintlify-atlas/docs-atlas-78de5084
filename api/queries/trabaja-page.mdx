## getTrabajaConNosotrosPageQuery

Retrieves all content for the "Work with Us" careers page including gallery, job offer search functionality, and CV submission section.

### Function Signature

```typescript
getTrabajaConNosotrosPageQuery(lang: string = 'es'): string
```

### Parameters

<ParamField path="lang" type="string" default="es">
  Language code for localized content. Supports 'es', 'ca', 'en', etc. Falls back to Spanish if translation not available.
</ParamField>

### GROQ Query

```groq
*[_type == "trabajaConNosotrosPage" && _id == "trabajaConNosotrosPage"][0] {
  "highlightedText": {
    "desktop": coalesce(highlightedText.desktop.ca, highlightedText.desktop.es, ""),
    "tablet": coalesce(highlightedText.tablet.ca, highlightedText.tablet.es, ""),
    "mobile": coalesce(highlightedText.mobile.ca, highlightedText.mobile.es, "")
  },
  "highlightedTextSecondary": {
    "tablet": coalesce(highlightedTextSecondary.tablet.ca, highlightedTextSecondary.tablet.es, ""),
    "mobile": coalesce(highlightedTextSecondary.mobile.ca, highlightedTextSecondary.mobile.es, "")
  },
  "bodyText": coalesce(bodyText.ca, bodyText.es, ""),
  "gallery": gallery[] {
    "url": asset->url,
    "alt": alt
  },
  "offersTitle": coalesce(offersTitle.ca, offersTitle.es, ""),
  "searchPlaceholder": coalesce(searchPlaceholder.ca, searchPlaceholder.es, ""),
  "loadMoreText": coalesce(loadMoreText.ca, loadMoreText.es, ""),
  "cvBackgroundImage": {
    "desktop": cvBackgroundImage.desktop.asset->url,
    "desktopAlt": cvBackgroundImage.desktop.alt,
    "tablet": cvBackgroundImage.tablet.asset->url,
    "tabletAlt": cvBackgroundImage.tablet.alt,
    "mobile": cvBackgroundImage.mobile.asset->url,
    "mobileAlt": cvBackgroundImage.mobile.alt
  },
  "cvHighlightedText": {
    "desktop": coalesce(cvHighlightedText.desktop.ca, cvHighlightedText.desktop.es, ""),
    "tablet": coalesce(cvHighlightedText.tablet.ca, cvHighlightedText.tablet.es, ""),
    "mobile": coalesce(cvHighlightedText.mobile.ca, cvHighlightedText.mobile.es, "")
  },
  "cvBodyText": coalesce(cvBodyText.ca, cvBodyText.es, "")
}
```

### Usage Example

```typescript
import { sanityClient } from '@/lib/sanity';
import { getTrabajaConNosotrosPageQuery } from '@/lib/sanityQueries';

// In an Astro page component
const lang = Astro.currentLocale || 'es';
const trabajaData = await sanityClient.fetch(getTrabajaConNosotrosPageQuery(lang));
```

```astro
---
// pages/trabaja-con-nosotros.astro
import { sanityClient } from '@/lib/sanity';
import { getTrabajaConNosotrosPageQuery } from '@/lib/sanityQueries';

const lang = Astro.currentLocale || 'es';
const data = await sanityClient.fetch(getTrabajaConNosotrosPageQuery(lang));
---

<section>
  <h1>{data.highlightedText.desktop}</h1>
  <p>{data.bodyText}</p>
  
  <div class="gallery">
    {data.gallery.map(image => (
      <img src={image.url} alt={image.alt} />
    ))}
  </div>
  
  <div class="job-offers">
    <h2>{data.offersTitle}</h2>
    <input 
      type="search" 
      placeholder={data.searchPlaceholder}
      id="job-search"
    />
    <div id="job-listings">
      <!-- Job listings populated dynamically -->
    </div>
    <button>{data.loadMoreText}</button>
  </div>
  
  <div 
    class="cv-submission"
    style={`background-image: url(${data.cvBackgroundImage.desktop})`}
  >
    <h2>{data.cvHighlightedText.desktop}</h2>
    <p>{data.cvBodyText}</p>
    <!-- CV upload form -->
  </div>
</section>
```

### Related Queries

This page works in conjunction with:
- `getJobOffersQuery()` - Fetches active job listings for the offers section
- `getProvincesQuery()` - Provides province filter options for job search

### Notes

- Job offers are fetched separately using `getJobOffersQuery()` for better performance
- The gallery array contains resolved URLs ready for display
- Responsive background images allow optimal loading for different devices
- Search and load more functionality requires client-side implementation
