### Province Markers

Markers are positioned absolutely using percentage coordinates:

```astro
{pizzeriasPageData.provinces.map((province) => {
  const hasPosition = province.mapPositionX !== null && 
                      province.mapPositionY !== null;
  
  return (
    <div
      class={`map-marker province-${province.slug}`}
      data-province={province.name}
      style={hasPosition ? 
        `left: ${province.mapPositionX}%; top: ${province.mapPositionY}%;` : ''}
    >
      {/* Marker icon */}
      <span class="marker-count">{province.locals?.length || 0}</span>
    </div>
  );
})}
```

**Data attributes:**
- `class`: `map-marker province-{slug}` for targeting
- `data-province`: Province name for display
- `style`: Inline positioning (left/top percentages)

## Map Interactivity

### Click-to-Scroll

**File:** `src/pages/pizzerias.astro:193-236`

Clicking a province marker scrolls to the corresponding section:

<Steps>

<Step title="Extract province slug from marker class">

```javascript
const classList = Array.from(marker.classList);
const provinceClass = classList.find(cls => cls.startsWith('province-'));
const slug = provinceClass.replace('province-', '');
```

</Step>

<Step title="Find target section">

```javascript
const targetSection = document.getElementById(`province-${slug}`);
```

</Step>

<Step title="Calculate scroll position with header offset">

```javascript
const targetPosition = targetSection.offsetTop - 90; // 70px header + 20px margin
```

</Step>

<Step title="Animate scroll with GSAP">

```javascript
gsap.to(window, {
  duration: 1.2,
  scrollTo: {
    y: targetPosition,
    autoKill: false
  },
  ease: "power3.inOut"
});
```

</Step>

<Step title="Add marker pulse effect">

```javascript
gsap.fromTo(marker,
  { scale: 1 },
  {
    scale: 1.3,
    duration: 0.2,
    yoyo: true,
    repeat: 1,
    ease: "power2.inOut"
  }
);
```

</Step>

</Steps>

### Sticky Map Behavior

**File:** `src/pages/pizzerias.astro:173-192`

On desktop (> 1000px), the map container pins to the viewport:

```javascript
function initMapSticky() {
  const mapContainer = document.querySelector('.map-container');
  const pizzeriasContainer = document.querySelector('.pizzerias-container');
  
  const mm = gsap.matchMedia();
  
  mm.add("(min-width: 1001px)", () => {
    if (mapContainer && pizzeriasContainer) {
      ScrollTrigger.create({
        trigger: pizzeriasContainer,
        start: 'top 55px',      // Start pinning when container reaches header
        end: 'bottom bottom',   // Unpin when container ends
        pin: mapContainer,
        pinSpacing: false,
        markers: false
      });
    }
  });
}
```

**Why 55px?** Accounts for the header height (typically 55-70px).

## Data Structure

### Province Object

```typescript
interface Province {
  name: string;           // "Valencia"
  slug: string;           // "valencia"
  mapPositionX: number;   // 45.5 (percentage)
  mapPositionY: number;   // 62.3 (percentage)
  mapIcon?: string;       // URL to custom marker icon
  mapIconAlt?: string;    // Alt text for icon
  mapIconColor?: string;  // Color for marker (if applicable)
  locals: Local[];        // Array of locations in this province
}
```

### Local Object

```typescript
interface Local {
  locality: string;       // "Centro Valencia"
  address?: string;       // "Calle Mayor, 123"
  location?: string;      // Google Maps URL
  phone?: string;         // "+34 123 456 789"
  availability: string[]; // ["delivery", "terraza", "pizzeria", "restaurante"]
}
```

### Full Page Data

```typescript
interface PizzeriasPageData {
  mapBackground?: string;    // Background image URL
  highlightedText?: string;  // Hero text on map
  subText?: string;          // Subtitle below map
  provinces: Province[];     // All provinces with locations
}
```

- Simplified map view

## Sanity Query Structure

**File:** `src/lib/sanityQueries.ts`

```groq
*[_type == "pizzeriasPage"][0] {
  mapBackground,
  highlightedText,
  subText,
  provinces[]->{
    name,
    "slug": slug.current,
    mapPositionX,
    mapPositionY,
    mapIcon,
    mapIconAlt,
    locals[]{
      locality,
      address,
      location,
      phone,
      availability[]
    }
  }
}
```

**Key points:**
- Provinces are referenced documents (`->`)
- Locals are embedded within provinces
- All text fields support localization
- Map positions are stored as numbers (0-100)

## Performance Optimizations

### Lazy Loading Images

All location images and icons load lazily:

```astro
<img src={image} alt={alt} loading="lazy" />
```

### Event Delegation

Marker click handlers use direct attachment for performance:

```javascript
markers.forEach((marker) => {
  marker.addEventListener('click', handleMarkerClick);
});
```

### ScrollTrigger Cleanup

ScrollTriggers are automatically cleaned up on SPA navigation via `astro:page-load` event.

### matchMedia

Sticky behavior only activates on desktop:

```javascript
gsap.matchMedia().add("(min-width: 1001px)", () => {
  // Desktop-only sticky logic
});
```

Prevents unnecessary calculations on mobile.
