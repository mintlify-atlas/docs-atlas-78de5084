# Welcome to Del Poble Pizzeria

Del Poble Pizzeria is a modern, server-rendered website built with Astro and powered by Sanity CMS. This project showcases a beautiful pizzeria website with full internationalization support for Spanish, English, and Valencian.

## What You'll Build

This project is a production-ready pizzeria website featuring:

- **🌍 Multi-language Support**: Full i18n with Spanish (default), English, and Valencian
- **📝 Headless CMS**: Sanity Studio for content management with localized fields
- **⚡ Blazing Fast**: Astro's server-side rendering for optimal performance
- **✨ Smooth Animations**: GSAP and Lenis for professional animations
- **📧 Contact Forms**: Server actions with Nodemailer integration
- **🎨 Modern UI**: Swiper for carousels and custom SCSS styling

## Tech Stack

Del Poble is built with modern web technologies:

- **[Astro](https://astro.build)** - Fast, server-first web framework
- **[Sanity](https://sanity.io)** - Headless CMS with real-time content
- **[Vercel](https://vercel.com)** - Deployment platform with edge functions
- **[GSAP](https://greensock.com/gsap/)** - Professional animation library
- **[Lenis](https://lenis.studiofreight.com/)** - Smooth scroll experience
- **[Swiper](https://swiperjs.com/)** - Modern touch slider

## Key Features

### Internationalization (i18n)

The site supports three languages with intelligent fallbacks:

- **Spanish (ES)** - Default language at `/`
- **English (EN)** - Available at `/en/`
- **Valencian (VAL)** - Available at `/val/`

<Note>
  Content is managed in Sanity with localized fields. Spanish is required as the fallback language, while English and Valencian are optional.
</Note>

### Server-Side Rendering

Configured with Astro's server output mode for:
- Dynamic content from Sanity
- Server actions for form handling
- SEO optimization
- Fast page loads

### Content Management

Sanity Studio provides:
- Real-time content editing
- Localized fields for each language
- Image optimization
- Portable text for rich content
