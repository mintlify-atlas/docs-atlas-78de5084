Get the Del Poble Pizzeria website up and running on your local machine.

<Steps>
  <Step title="Clone and install dependencies">
    Clone the repository and install all required packages:

    ```bash
    git clone <repository-url>
    cd source
    npm install
    ```
  </Step>

  <Step title="Configure environment variables">
    Copy the example environment file and configure your SMTP settings:

    ```bash
    cp .env.example .env
    ```

    Edit `.env` with your SMTP credentials:

    ```bash
    # SMTP Configuration for email sending
    SMTP_HOST=smtp.tuhosting.com
    SMTP_PORT=465
    SMTP_USER=noreply@tudominio.com
    SMTP_PASS=tu_contraseña_smtp
    EMAIL_TO=admin@tudominio.com
    ```

    <Note>
      SMTP credentials are provided by your hosting provider. Port 465 is for SSL, 587 for TLS.
    </Note>
  </Step>

  <Step title="Start the development server">
    Launch the Astro development server:

    ```bash
    npm run dev
    ```

    The site will be available at `http://localhost:4321`
  </Step>

  <Step title="Test language routing">
    The site supports three languages with different URL patterns:

    - **Spanish (default)**: `http://localhost:4321/`
    - **English**: `http://localhost:4321/en/`
    - **Valencian**: `http://localhost:4321/val/`

    Spanish is the default locale and doesn't require a URL prefix.
  </Step>

  <Step title="Explore available routes">
    Navigate to any of these pages in your browser:

    **Main Pages** (available in all languages):
    - `/` - Home page
    - `/historia` - History page
    - `/carta` - Menu page
    - `/pizzerias` - Locations page
    - `/talento` - Jobs/careers page
    - `/contacto` - Contact page
    - `/[slug]` - Dynamic general pages

    **Example URLs**:
    ```
    http://localhost:4321/historia           (Spanish)
    http://localhost:4321/en/historia        (English)
    http://localhost:4321/val/historia       (Valencian)
    ```
  </Step>

  <Step title="Set up Sanity Studio (optional)">
    The project uses Sanity CMS for content management. To run the Studio locally:

    ```bash
    cd delpoble-studio
    npm install
    npm run dev
    ```

    The Sanity Studio will be available at `http://localhost:3334`

    **Sanity Configuration**:
    - **Project ID**: `v8qkjh2b`
    - **Dataset**: `production`
    - **Studio Port**: `3334`

    The Astro frontend is already configured to fetch from this Sanity project (see `src/lib/sanity.ts:6-9`).
  </Step>
</Steps>

## Available Commands

All commands run from the project root:

| Command | Action |
| --- | --- |
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview production build locally |
| `npm run astro` | Run Astro CLI commands |

## Project Configuration

The site uses these key configurations:

**Output Mode**: Server-side rendering (SSR)  
**Adapter**: Vercel  
**i18n Locales**: `es` (default), `en`, `val`

See `astro.config.mjs:9-16` for the complete configuration.
